April 24, 2026
Feature Infrastructure

Multi-tenant accounts

Organic is now fully multi-tenant. Every user belongs to an account, and all data — companies, articles, clusters — is scoped to that account via Supabase Row Level Security policies.

  • Account-scoped RLS — All API routes and database queries filter by the authenticated user's account membership. No cross-tenant data leakage.
  • SECURITY DEFINER helpers — user_account_ids() and is_platform_admin() functions eliminate recursive RLS policy issues.
  • Role-gated navigation — Owners see the full Companies editor; scoped members see a read-only brand portal.

Full Stripe integration for subscription management, usage-based billing, and self-service account controls.

  • Subscription tiers — Starter, Standard, and Scale plans with configurable article limits and overage pricing.
  • Webhook processing — Real-time event handling for subscription changes, invoice payments, and cancellations.
  • Customer portal — Self-service billing management, plan upgrades, and payment method updates via Stripe's hosted portal.

MCP create_article

The Organic Brand MCP server now includes a create_article tool — the 21st tool in the unified server. MCP clients can now generate articles using the prompt-building tools and save the results directly to the Organic database without needing the web app.

  • End-to-end MCP workflow — Use get_blog_system_prompt + get_user_prompt_template to generate, then create_article to persist.

Feature Infrastructure

Unified MCP Server

Organic now exposes your complete brand toolkit as a single, unified Model Context Protocol (MCP) server — replacing the original three separate servers with one install, 20 tools.

  • Organic Brand MCP — One server with 20 tools spanning voice (5), photo style (6), blog (8), and general (1).
  • Simplified setup — Single entry in your Claude Desktop or Cursor MCP config instead of three.
  • Backward-compatible — All tool names are identical to the original servers. No client-side changes needed.

Compatible with Claude Desktop, Cursor, ChatGPT, Windsurf, and any MCP-enabled AI client. Update a voice profile in Organic → every connected client sees the change instantly.

Feature Infrastructure

Fact-check consul

Introduced the multi-model Fact-Check Consul. Every factual claim is independently verified by Gemini and Grok, with a synthesized consensus report showing agreement level, individual model reasoning, linked sources, and one-click suggested rewrites.

  • Dual-model verification — Gemini 3.1 Pro and Grok cross-verify every claim independently.
  • Agreement scoring — Full Agreement, Partial Agreement, Models Disagree, or Single Source.
  • Inline corrections — One-click "Apply Fix" buttons update article content directly from the consul report.
Why two models Single-model fact-checking produces confident-sounding but sometimes wrong verdicts. Two independent models catch each other's blind spots — when they disagree, it flags human review.
